#### 技术栈:
react + redux + react-router 4 + ES6/7 + Immutable + axios

#### 运行项目:
> git clone https://github.com/yuxiluo/react-jianshu.git  
> cd react-jianshu  
> npm i 或 cnpm install  
> npm start   
> npm run build (打包)
